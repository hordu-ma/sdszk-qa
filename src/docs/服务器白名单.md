# A/B 服务器出站白名单（生产部署）

> 基础设施与部署请参考：`src/infra/README.md`。

> 适用场景：A 为入口层（Nginx + 前端静态），B 为核心层（FastAPI + vLLM + PostgreSQL + MinIO）。
> 说明：A 服务器为 `172.18.6.117`（共享现网 Nginx，仅上传前端文件）；B 服务器为 `172.18.6.123`（核心层专用）。

## 一、B 服务器（核心层）

### 1. 系统与容器安装（APT + Docker + NVIDIA）

**出站端口：TCP 80/443**

- Ubuntu APT 源（国内镜像，任选其一即可）：
  - mirrors.tuna.tsinghua.edu.cn
- Docker APT 源（国内镜像）：
  - mirrors.tuna.tsinghua.edu.cn
- NVIDIA 容器工具链：
  - nvidia.github.io
  - packages.nvidia.com

### 2. 镜像拉取（Docker 国内镜像）

**出站端口：TCP 443**

- docker.mirrors.ustc.edu.cn

> 若使用其它国内镜像，请替换为对应域名。

### 3. 模型拉取（ModelScope）

**出站端口：TCP 443**

- modelscope.cn
- www.modelscope.cn
